    3. Create `internal/metadata/tag.go`:
       - `func extractWithTag(filePath string) (*BookMetadata, error)`:
         - `os.Open(filePath)`, defer f.Close() immediately (NAS file handle concern from research).

         - Map: Title from `m.Title()`, Author from `m.AlbumArtist()` (falls back to `m.Artist()` if empty), Genre from `m.Genre()`, Year from `m.Year()`.
