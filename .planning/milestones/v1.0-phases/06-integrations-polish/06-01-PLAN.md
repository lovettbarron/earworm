       ```go
       var csvHeader = []string{
           "Title", "Author", "ISBN", "ISBN13", "My Rating",
           "Average Rating", "Publisher", "Year Published",
           "Date Read", "Date Added", "Bookshelves", "Exclusive Shelf",
       }
       ```

    3. `ExportCSV(w io.Writer, books []db.Book) error`:
       - Create csv.NewWriter(w), defer Flush()
       - Write header row
       - For each book:
         - Skip if book.Title == "" or book.Author == "" (log slog.Warn with ASIN)
         - Build record slice:
           - [0] Title: b.Title
           - [1] Author: b.Author
           - [2] ISBN: "" (Audible has no ISBN)
           - [3] ISBN13: ""
           - [4] My Rating: ""
           - [5] Average Rating: ""
           - [6] Publisher: ""
           - [7] Year Published: formatYear(b.Year) -- "2021" if > 0, else ""
           - [8] Date Read: reformatDate(b.PurchaseDate) -- convert "2024-01-15" to "2024/01/15"
           - [9] Date Added: b.CreatedAt.Format("2006/01/02")
           - [10] Bookshelves: "read, audiobook" (per D-08)
           - [11] Exclusive Shelf: "read" (per D-08)
         - Write record
       - Return cw.Error()

    4. Helper `reformatDate(iso string) string`:
       - If iso == "", return ""
       - Replace "-" with "/" using strings.ReplaceAll
       - Return result (converts "2024-01-15" to "2024/01/15")

    5. Helper `formatYear(year int) string`:
       - If year <= 0, return ""
       - Return fmt.Sprintf("%d", year)
