    Rewrite README.md with these sections in order (per D-14, D-15):

    **1. Header and description:**
    - `# Earworm` heading
    - One-paragraph description: CLI-driven audiobook library manager for Audible. Downloads, organizes into Audiobookshelf-compatible structure, triggers library scans. Single Go binary.

    **2. Features** (bullet list):
    - SQLite-backed library state tracking
    - Audible library sync via audible-cli
    - Fault-tolerant batch downloads with rate limiting and crash recovery
    - Automatic organization into Author/Title [ASIN]/ folder structure
    - Audiobookshelf library scan integration
    - Goodreads CSV export
    - Daemon/polling mode for unattended operation
    - Cross-filesystem file moves (local to NAS)

    **3. Prerequisites:**
    - Go 1.23+ (building from source)
    - Python 3.9+ (required by audible-cli -- managed automatically by earworm)
    - Note: audible-cli is installed automatically into an embedded venv on first use

    **4. Installation:**
    - From source: `go install github.com/lovettbarron/earworm/cmd/earworm@latest`
    - From releases: download from GitHub releases page
    - Verify: `earworm version`

    **5. Quickstart** (per D-15, step-by-step):
    ```
    # 1. Install earworm
    go install github.com/lovettbarron/earworm/cmd/earworm@latest

    # 2. Initialize configuration
    earworm config init

    # 3. Set your library path (where audiobooks are stored)
    earworm config set library_path /path/to/audiobooks

    # 4. Authenticate with Audible
    earworm auth

    # 5. Sync your Audible library
    earworm sync

    # 6. Preview what would be downloaded
    earworm download --dry-run

    # 7. Download new books
    earworm download

    # 8. Organize into library structure
    earworm organize
    ```

    **6. Commands** (full reference, each with description, usage, flags, example):

    - `earworm auth` -- Authenticate with Audible via audible-cli
    - `earworm sync` -- Sync Audible library metadata to local database. Flags: `--json`
    - `earworm scan` -- Scan local library directory for existing audiobooks. Flags: `--recursive`
    - `earworm status` -- Display library status. Flags: `--json`, `--author`, `--status`
    - `earworm download` -- Download new audiobooks. Flags: `--dry-run`, `--json`, `--limit N`, `--asin ASIN`
    - `earworm organize` -- Organize downloads into library structure. Flags: `--json`
    - `earworm notify` -- Trigger Audiobookshelf library scan. Flags: `--json`
    - `earworm goodreads` -- Export library to Goodreads CSV. Flags: `--output FILE` / `-o FILE`
    - `earworm daemon` -- Run in polling mode. Flags: `--interval DURATION`, `--verbose`, `--once`
    - `earworm config init` -- Create default config file
    - `earworm config show` -- Show current configuration
    - `earworm config set KEY VALUE` -- Update a setting
    - `earworm version` -- Display version information

    Global flags: `--config FILE`, `--quiet` / `-q`

    **7. Configuration:**
    Table of all config keys with defaults and descriptions:
    | Key | Default | Description |
    |-----|---------|-------------|
    | `library_path` | (none) | Path to audiobook library (required) |
    | `staging_path` | `~/.config/earworm/staging` | Temporary download directory |
    | `audible_cli_path` | `audible` | Path to audible-cli binary |
    | `audible.profile_path` | (none) | Path to audible-cli profile |
    | `audiobookshelf.url` | (none) | Audiobookshelf server URL |
    | `audiobookshelf.token` | (none) | Audiobookshelf API token |
    | `audiobookshelf.library_id` | (none) | Audiobookshelf library ID |
    | `daemon.polling_interval` | `6h` | Polling interval for daemon mode |
    | `download.rate_limit_seconds` | `5` | Seconds between download requests |
    | `download.max_retries` | `3` | Max retry attempts per book |
    | `download.backoff_multiplier` | `2.0` | Backoff multiplier for retries |
    | `scan.recursive` | `false` | Scan subdirectories recursively |

    Config file location: `~/.config/earworm/config.yaml`

    **8. Audiobookshelf Integration:**
