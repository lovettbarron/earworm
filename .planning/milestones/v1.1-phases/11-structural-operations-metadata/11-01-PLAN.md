    1. Result types:
    ```go
    type FlattenResult struct {
        BookDir     string
        FilesMoved  []FileMoveResult
        DirsRemoved []string
        Errors      []error
    }

    type FileMoveResult struct {
        SourcePath string
        DestPath   string

    }
    ```

    2. `FlattenDir(bookDir string) (*FlattenResult, error)`:
       a. Initialize `FlattenResult{BookDir: bookDir}`
       b. Use `filepath.WalkDir(bookDir, ...)` to find all files in subdirectories
       c. Skip files where `filepath.Dir(path) == bookDir` (already at root level)
       d. Match audio extensions case-insensitively: `.m4a`, `.m4b` (same as FindAudioFiles)
       e. For each nested audio file, compute destination as `filepath.Join(bookDir, filepath.Base(path))`
       f. Call `uniquePath(destPath)` to handle name collisions -- if dest exists, try `name_1.ext`, `name_2.ext`, etc.
       g. Call `VerifiedMove(path, destPath)` -- on success, record FileMoveResult with SHA256 hash and Success=true; on error, record with Success=false and Error string, continue processing (error isolation per research Pitfall 6)
       h. After walk, call `removeEmptyDirs(bookDir)` to clean up bottom-up

    3. `uniquePath(path string) string` (unexported):
       - If path does not exist, return it as-is
       - Otherwise, split into name+ext, try `name_1.ext`, `name_2.ext`, etc. up to 999
       - Return first non-existing path

    4. `removeEmptyDirs(root string) []string` (unexported):
       - Walk the directory tree, collect all subdirectories (not root itself)
       - Sort by depth descending (deepest first)
       - For each, try `os.Remove` (only succeeds if empty)
       - Return list of removed directory paths
