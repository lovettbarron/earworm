  <action>
    1. In internal/db/plans.go: Export `isValidOpType` by renaming to `IsValidOpType`. The existing unexported function becomes:
       ```go
       // IsValidOpType checks whether an operation type string is in the allowed set.
       func IsValidOpType(opType string) bool {
           for _, t := range ValidOpTypes {
               if t == opType {
                   return true
               }
           }
           return false
       }
       ```
       Update any internal callers of `isValidOpType` (in AddOperation) to use `IsValidOpType`.

    2. Create internal/planengine/csvimport.go with:
       - `CSVRowError` struct: `Line int`, `Column string`, `Message string`
       - `CSVImportResult` struct: `PlanID int64`, `RowCount int`, `ErrorCount int`, `Errors []CSVRowError`
       - `StripBOM(r io.Reader) io.Reader` function: peek first 3 bytes, discard if 0xEF 0xBB 0xBF
       - `ImportCSV(database *sql.DB, planName string, r io.Reader) (*CSVImportResult, error)` function:
         a. Call StripBOM on reader
         b. Create csv.Reader with TrimLeadingSpace=true
         c. Read header row, build case-insensitive column index map (lowercase keys)
         d. Validate required columns: "op_type", "source_path" (error if missing)
         e. Read each data row, validate:
            - op_type via db.IsValidOpType (error if invalid)
            - source_path non-empty (error if empty)
            - dest_path non-empty when op_type is "move" or "flatten" (error if empty)
         f. Line number tracking: header=line 1, first data row=line 2, etc.
         g. If any errors: return CSVImportResult with Errors populated, PlanID=0, no plan created
         h. If no errors: call db.CreatePlan(database, planName, "Imported from CSV (N operations)")
            then db.AddOperation for each row with Seq starting at 1
         i. Return CSVImportResult with PlanID, RowCount set

    3. Create internal/planengine/csvimport_test.go with tests from behavior block above.
