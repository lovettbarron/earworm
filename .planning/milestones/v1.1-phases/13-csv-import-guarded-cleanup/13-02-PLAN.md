    3. Create internal/planengine/cleanup.go with:
       - `MoveToTrash(sourcePath, trashDir string) (trashPath string, err error)`:
         a. Build unique trash path: `trashDir/<unixnano>_<basename>`
         b. `os.MkdirAll(trashDir, 0755)`
         c. Try `os.Rename(sourcePath, trashPath)`
         d. On EXDEV error: fall back to copy (io.Copy) + verify size matches + delete source
         e. Return the trashPath for audit logging

       - `CleanupResult` struct: `Moved int`, `Skipped int`, `Errors []string`

       - `CleanupExecutor` struct: `DB *sql.DB`, `TrashDir string`

       - `func (c *CleanupExecutor) ListPending(planID int64) ([]db.PlanOperation, error)`:
         calls `db.ListDeleteOperations(c.DB, "completed", planID)`, filters to ops with status "pending" or "completed" (completed plan ops that represent delete actions to perform)

       - `func (c *CleanupExecutor) Execute(ops []db.PlanOperation) (*CleanupResult, error)`:
         a. For each op:
            - Check if source_path exists on disk (os.Stat)
            - If missing: increment Skipped, log warning, update op status to "skipped" with error "file not found", continue
            - Call MoveToTrash(op.SourcePath, c.TrashDir)
            - On success: update op status to "completed", log audit with entity_type="operation", entity_id=strconv.FormatInt(op.ID, 10), action="cleanup_trash", after_state=JSON{"trash_path": trashPath}
            - On failure: increment error count, log audit with success=false, continue (don't abort batch)
         b. Return CleanupResult
