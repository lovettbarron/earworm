    - EnsureAudibleCLI creates venv via `python3 -m venv {dir}` if dir doesn't exist
    - EnsureAudibleCLI runs `{venv}/bin/pip install audible-cli` if audible binary missing
    - EnsureAudibleCLI returns error if python3 is not found on PATH
    - EnsureAudibleCLI returns the resolved audible binary path on success
  </behavior>
  <action>

    1. `VenvDir() (string, error)` — returns `filepath.Join(config.ConfigDir(), "venv")`.

    3. `EnsureAudibleCLI(ctx context.Context, w io.Writer) (string, error)` — the main bootstrap function:
       - Call `AudibleCLIPath()` to get expected binary path.

       - Run `{venvDir}/bin/pip install --upgrade audible-cli`. Write progress to `w` ("Installing audible-cli..."). Capture stderr for error reporting.
       - Verify the audible binary now exists. Return its path.

    - Test EnsureAudibleCLI fast path: create a temp dir mimicking venv structure with a fake executable audible binary, verify it returns immediately without running pip.
    - Test EnsureAudibleCLI error when python3 not found (inject execCommand that fails LookPath).
