**Step 5: Create `internal/db/migrations/001_initial.sql`.**

Per D-09, initial schema with books table (ASIN primary key) and indexes:
```sql
CREATE TABLE IF NOT EXISTS books (
    asin TEXT PRIMARY KEY,
    title TEXT NOT NULL DEFAULT '',
    author TEXT NOT NULL DEFAULT '',
    status TEXT NOT NULL DEFAULT 'unknown',
    local_path TEXT NOT NULL DEFAULT '',
    created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX IF NOT EXISTS idx_books_status ON books(status);
```

Note: `schema_versions` table is created programmatically in db.go, not in the migration file.

**Step 6: Create `internal/db/db.go`.**

Implement per the Pattern 3 (Embedded SQL Migrations) from RESEARCH.md:

- `Open(dbPath string) (*sql.DB, error)` -- opens SQLite with `sql.Open("sqlite", dbPath)` (NOT "sqlite3" per Pitfall 1), sets WAL mode, runs migrations
- `runMigrations(db *sql.DB) error` -- creates `schema_versions` table, reads embedded `.sql` files sorted by name, parses version number from filename prefix (e.g. `001_initial.sql` -> version 1), skips already-applied versions, executes each in a transaction and records in schema_versions
- Import `_ "modernc.org/sqlite"` for driver registration
- Use `//go:embed migrations/*.sql` for embedded migrations

Per D-08, the schema_versions table: `version INTEGER PRIMARY KEY, applied_at DATETIME DEFAULT CURRENT_TIMESTAMP`.

**Step 7: Create `internal/db/books.go`.**

Define `Book` struct and CRUD functions operating on `*sql.DB`:

```go
type Book struct {
    ASIN      string
    Title     string
    Author    string
    Status    string
    LocalPath string
    CreatedAt time.Time
    UpdatedAt time.Time
}
```

Functions:
- `InsertBook(db *sql.DB, book Book) error` -- INSERT into books, fails on duplicate ASIN (PRIMARY KEY constraint)
- `GetBook(db *sql.DB, asin string) (*Book, error)` -- SELECT by ASIN, return nil if not found (no error for missing)
- `ListBooks(db *sql.DB) ([]Book, error)` -- SELECT all, ordered by created_at DESC
- `UpdateBookStatus(db *sql.DB, asin string, status string) error` -- UPDATE status and updated_at WHERE asin matches, return error if no rows affected

Valid status values (validate in Go, not DB): `unknown`, `scanned`, `downloading`, `downloaded`, `organized`, `error`.
