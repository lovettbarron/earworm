3. Update `init()` to register subcommand and new flags:
```go
func init() {
    scanCmd.Flags().BoolVarP(&scanRecursive, "recursive", "r", false, "recursively scan nested directories")
    scanCmd.Flags().BoolVar(&scanDeep, "deep", false, "scan all folders including those without ASINs and detect issues")
    scanCmd.Flags().BoolVar(&scanJSON, "json", false, "output in JSON format (only with --deep)")

    scanIssuesCmd.Flags().BoolVar(&scanIssuesJSON, "json", false, "output in JSON format")
    scanIssuesCmd.Flags().BoolVar(&scanCreatePlan, "create-plan", false, "create a plan from detected issues")
    scanIssuesCmd.Flags().StringVar(&scanFilterType, "type", "", "filter issues by type")
    scanCmd.AddCommand(scanIssuesCmd)

    rootCmd.AddCommand(scanCmd)
}
```

4. Add `runScanIssues` function that:
   - Opens DB via `config.DBPath()` and `db.Open()`
   - Calls `db.ListScanIssuesByType` if `scanFilterType != ""`, else `db.ListScanIssues`
   - If `scanCreatePlan` is set, calls `scanner.CreatePlanFromIssues(database, issues)`, prints plan ID and counts, returns
   - If `scanIssuesJSON` is set, uses `json.NewEncoder(cmd.OutOrStdout()).SetIndent("", "  ")` to encode issues
   - Otherwise prints human-readable table: one line per issue with `fmt.Fprintf(cmd.OutOrStdout(), "  %-12s %-8s %s\n    %s\n", issue.IssueType, issue.Severity, issue.Path, issue.Message)`
   - Prints total count at end

5. Add JSON output type for deep scan:
```go
type deepScanJSON struct {
    TotalDirs   int            `json:"total_dirs"`
    WithASIN    int            `json:"with_asin"`
    WithoutASIN int            `json:"without_asin"`
    IssuesFound int            `json:"issues_found"`
    IssueCounts map[string]int `json:"issue_counts"`
    Issues      []scanIssueJSON `json:"issues"`
}

type scanIssueJSON struct {
    ID              int64  `json:"id"`
    Path            string `json:"path"`
    IssueType       string `json:"issue_type"`
    Severity        string `json:"severity"`
    Message         string `json:"message"`
    SuggestedAction string `json:"suggested_action"`
}
```

6. Modify `runDeepScan` to accept `scanJSON` flag. After `scanner.DeepScanLibrary` completes:
   - If `scanJSON` is set, query `db.ListScanIssues(database)` to get full issues list, build `deepScanJSON` struct (convert `IssueCounts` keys from `scanner.IssueType` to `string`), encode with `json.NewEncoder(cmd.OutOrStdout()).SetIndent("", "  ")`, return
   - Otherwise continue with existing human-readable output (no change)
